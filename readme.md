## Quick Start

### Prerequisites

- **Node.js** (v14+)
- **NPM** (v6+)
- **Git**

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/yourusername/dhanesh-portfolio.git
cd dhanesh-portfolio
```

2. **Install dependencies**

```bash
npm install
```

3. **Start development server**

```bash
npm start
```

4. **Open in browser**

```
http://localhost:3000
```
